**Generating a Simple Program with IF-ELSE LOOP using Python**

```python
# Define a function that asks for user input and calculates the result based on the condition
def calculate_grade(score):
    # If the score is greater than 90, print 'A' (Excellent)
    if score > 90:
        return 'A'
    # If the score is between 80 and 90, print 'B' (Good)
    elif score >= 80:
        return 'B'
    # If the score is between 70 and 79, print 'C' (Fair)
    elif score >= 70:
        return 'C'
    # If the score is between 60 and 69, print 'D' (Poor)
    elif score >= 60:
        return 'D'
    # If the score is less than 60, print 'F' (Fail)
    else:
        return 'F'

# Ask for user input
score = int(input("Enter your exam score: "))

# Calculate and print the result
result = calculate_grade(score)
print(f"Your grade is: {result}")
```

To explain this code line by line:

1. `def calculate_grade(score):`
   - This line defines a function called `calculate_grade` that takes one argument, `score`.
   - The purpose of this function is to determine the student's grade based on their score.

2. `if score > 90:`
   - If the condition inside the `if` statement (`score > 90`) is true, it means the score is greater than 90.
   - In this case, the code within the `if` block will be executed.

3. `return 'A'`
   - When the score is above 90, the function returns the string `'A'`, indicating an 'Excellent' grade.

4. The rest of the function follows similar lines to handle scores between different ranges:
   - Each range has its corresponding return statement with a specific letter grade (e.g., `return 'B' for scores >= 80`).

5. The last line in the function handles scores below 60:
   - In this case, the function returns `'F'`, indicating a 'Fail'.

6. `score = int(input("Enter your exam score: "))`
   - This line asks the user to input their score.
   - The `input()` function returns a string, so we use `int()` to convert it into an integer for calculations.

7. `result = calculate_grade(score)`
   - We call our function with the score entered by the user and store the result in the variable `result`.

8. `print(f"Your grade is: {result}")`
   - Finally, we print out the student's grade to the console using an f-string.

This code provides a basic example of how IF-ELSE loops can be used in Python programming to make decisions based on conditions and execute different blocks of code accordingly.
